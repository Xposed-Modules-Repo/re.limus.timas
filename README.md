TAssistant 是一个主要目的以促进 TIM体验 为主的 Xposed 模块

## 使用方法

在 Xposed框架 中激活本模块后，在 TIM 自带设置中点击 TAssistant 即可进入设置页面

- Android >= 8.1
- TIM >= 4.0.95 (即TIM_NT)
- Architecture = `arm64-v8a`
